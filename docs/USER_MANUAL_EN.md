## 1. Introduction

### What is Agentic Engine?

The Agentic Engine is an intelligent AI system that helps manage your store automatically with these capabilities:

- **Learn from customer behavior** - Remembers customer preferences and recommends suitable products
- **Make automatic decisions** - Analyzes situations and makes decisions for you
- **Self-improvement** - Gets smarter the more you use it
- **Smart notifications** - Alerts you when attention is needed

### System Features

| Feature | Description |
|---------|-------------|
| 🎯 Auto Decision | Approve/reject orders based on conditions |
| 📊 Data Analysis | Summarize sales, customers, trends |
| 🔔 LINE Notifications | Send messages when new orders arrive |
| 💡 Product Recommendations | Recommend products based on preferences |
| 📦 Inventory Management | Alert when stock is low |

## 4. Order Management

### Order Process

```
1. Customer places order → 2. System receives → 3. AI checks → 4. Kitchen → 5. Delivery
```

## 8. Troubleshooting

### ❌ Cannot access website

**Solution:**
1. Check internet connection
2. Try opening new browser
3. Try from another device

### ❌ Not receiving LINE notifications

**Solution:**
1. Verify Access Token
2. Check LINE Official Account
3. Verify Webhook settings

### ❌ Orders not showing

**Solution:**
1. Refresh page
2. Check API connection
3. Contact support
