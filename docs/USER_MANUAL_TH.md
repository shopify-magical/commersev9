## 1. แนะนำระบบ

### 🤖 Agentic Engine คืออะไร?

ระบบ Agentic Engine เป็นระบบ AI อัจฉริยะที่ช่วยจัดการร้านค้าอัตโนมัติ โดยมีความสามารถ:

- **เรียนรู้จากพฤติกรรมลูกค้า** - ระบบจะจดจำความชอบของลูกค้าและแนะนำสินค้าที่เหมาะสม
- **ตัดสินใจอัตโนมัติ** - วิเคราะห์สถานการณ์และตัดสินใจแทนคุณ
- **ปรับปรุงตัวเอง** - ยิ่งใช้งาน ยิ่งฉลาดขึ้น
- **แจ้งเตือนอัจฉริยะ** - แจ้งเตือนเมื่อมีออเดอร์ใหม่ หรือต้องการความสนใจ

### สิ่งที่ระบบทำได้

| ฟีเจอร์ | รายละเอียด |
|--------|-----------|
| 🎯 ตัดสินใจอัตโนมัติ | อนุมัติ/ปฏิเสธออเดอร์ตามเงื่อนไข |
| 📊 วิเคราะห์ข้อมูล | สรุปยอดขาย ลูกค้า แนวโน้ม |
| 🔔 แจ้งเตือน LINE | ส่งข้อความเมื่อมีออเดอร์ใหม่ |
| 💡 แนะนำสินค้า | แนะนำสินค้าตามความชอบลูกค้า |
| 📦 จัดการสต็อก | แจ้งเตือนเมื่อสินค้าใกล้หมด |

## 8. การแก้ปัญหา

### ❌ ไม่สามารถเข้าถึงเว็บไซต์

**วิธีแก้:**
1. ตรวจสอบอินเทอร์เน็ต
2. ลองเปิดเบราว์เซอร์ใหม่
3. ลองเข้าผ่านเครื่องอื่น

### ❌ ไม่ได้รับแจ้งเตือน LINE

**วิธีแก้:**
1. ตรวจสอบ Access Token
2. ตรวจสอบ LINE Official Account
3. ตรวจสอบการตั้งค่า Webhook

### ❌ ออเดอร์ไม่ขึ้น

**วิธีแก้:**
1. รีเฟรชหน้าจอ
2. ตรวจสอบการเชื่อมต่อ API
3. ติดต่อทีมสนับสนุน
